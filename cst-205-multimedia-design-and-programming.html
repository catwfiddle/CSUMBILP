Rose-colored glass

​This is a rose colored glass filter that allows the picture to have a rose colored filter. This function takes in a picture, then gets the pixels from the photo. Then for each of the pixels in the given photo is changed. The red pixels in the photo is incremented the most amount and the blue and green pixels in the photo is also increased.

def roseColoredGlasses():

   pic = get_pic()

   pixels = getPixels(pic)

   for p in pixels:

     r = getRed(p)

     b = getBlue(p)

     g = getGreen(p)

     setRed(p, r*1.6)

     setGreen(p, g*0.1)

     setBlue(p, b*1)

Negative

The make negative filter turns the photo negative. This function takes a picture, then gets all the pixels from the picture. Then this function  decreases the red color pixel values, blue color pixel values and green color pixel values in the picture from the 255. Then repaints the picture to the new pixel values.

 

def makeNegative():

    pic = get_pic()

    pixels = getPixels(pic)

    for p in pixels:

        r = 255 - getRed(p)

        b = 255 - getBlue(p)

        g = 255 - getGreen(p)

        setRed(p, r)

       setGreen(p, g)

       setBlue(p, b)

       repaint(pic)

def whiteCopy(src, target, targetX, targetY):

width = getWidth(src)

height = getHeight(src)

#prevent image from going over the edge

if targetY < 0: targetY = 0

if targetX < 0: targetX = 0

if targetX+width >= getWidth(target): width = getWidth(target)-targetX-1

if targetY+height >= getHeight(target): height = getHeight(target)-targetY-1

for x in range (0, width):

for y in range (0, height):

p = getPixel(src, x, y)

if distance(makeColor(255, 225, 255), getColor(p)) > 40:

setColor(getPixel(target, targetX+x, targetY+y), getColor(getPixel(src, x, y)))

return target

def lineDrawing():

  file = pickAFile()

  pic = makePicture(file)

  width = getWidth(pic)

  height = getHeight(pic)

  for x in range(0, width-1):

    for y in range (0, height-1):

      p = getPixel(pic, x, y)

      px = lum(getPixel(pic, x, y))

      right = lum(getPixel(pic, x+1, y))

      below = lum(getPixel(pic, x, y+1))

      if abs(px-right)>10:

         setColor(p, black)

      else:

          setColor(p, white)

      if abs(px-below)>10:

           setColor(p, black)

      else:

          setColor(p, white)

 







Line drawing:

This function turns the picture into a black and white outline of the picture.
With the JES IDE, the picture gets repainted starting from top left index  to right index.
The lineDrawing function gets the total height and width of the picture that is passed in.
Then it repaints the picture starting from the range position of index 0 from the left side to the
right side in the position of the total width of the picture -1 and the total height of the picture -1.
Then, this function uses the luminance ratio (r*0.299 + g*0.587 + b*0.114) which is used to convert any
picture into a gray scale picture to convert the pixels into a gray scale pixel.
Then, this function dose a comparison of the pixel values in the picture.
It compares if there is a difference between the starting pixel's value in the picture to the
value of the pixel on right side of it and the starting pixel's value and the value of the pixel below it.
If there is a difference between these pixels values, then it will repaint the starting pixel black.
However, if there isn't a  difference between the starting pixel's value and the value of the pixel on
right and the starting pixel's value and the pixel's value below it, it will paint the starting pixel white.
